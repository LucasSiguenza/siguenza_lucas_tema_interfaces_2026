# MjorApp-2025
### Descripción general
Aplicación desarrollada para comercios de comida. El proyecto busca optimizar la operatividad interna del restaurante y elevar la experiencia del cliente, ofreciendo una solución innovadora, robusta y adaptable a cualquier entorno. Está compuesto desarrollada con Ionic 7, Angular 20, y Supabase como base de datos.
